# **App Name**: AutoInterview

## Core Features:

- Avatar Creation & Management: Dashboard for creating, configuring, and managing AI interview avatars. Admins can set roles, difficulty, question types, and evaluation criteria.
- Automated Interview Scheduling: Generate unique interview links, send invites, track status, and set expiration times, all automated via the system.
- AI Interview Conductor: Use Claude API to conduct intelligent, role-specific interviews with adaptive follow-up questions and hint provision using a tool. Code submissions are executed and tested live.
- Real-time Interview Room: Provide candidates with a distraction-free live interview room including integrated code editor and auto-save capabilities.
- Evaluation & Scoring System: Implement a multi-dimensional scoring system (Technical, Communication, Problem-Solving, Culture Fit) to assess candidates and provide a comprehensive hiring recommendation using an LLM.
- Hiring Manager Review Dashboard: Enable real-time notifications, interview summaries, full transcripts, code submissions, and side-by-side candidate comparison for efficient review and decisions.
- Question Banks and Knowledge Retrieval: Offer pre-loaded, role-specific question banks with company-specific knowledge base integration, leveraging context-aware question generation based on past interviews.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey expertise and professionalism.
- Background color: Light grey (#F5F5F5) for a clean, modern look.
- Accent color: Teal (#009688) to highlight key actions and important information.
- Body text: 'Inter', sans-serif. Headline font: 'Space Grotesk', sans-serif. For code snippets: 'Source Code Pro', monospace.
- Use a set of clear, consistent icons (feather or material design) related to interviews and candidate evaluation.
- Maintain a clean and intuitive layout with clear visual hierarchy for easy navigation.
- Subtle transitions and animations for user interactions, such as loading indicators and confirmation messages.
